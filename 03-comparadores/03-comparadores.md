- 03. Comparadores
    - 👩🏻‍🏫 Código feito na aula
        
        ```jsx
        // Comparadores => retornar para nós um true ou false
        const num1 = 5
        const num2 = 15
        
        console.log(num1 > num2)
        console.log(num1 < num2)
        console.log(num1 >= num2)
        console.log(num1 <= num2)
        console.log(num1 === num2)
        console.log(num1 !== num2)
        ```
